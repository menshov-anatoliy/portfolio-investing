# portfolio-investing

Репозиторий для построения инвестиционных портфелей с использованием современных количественных методов (Hierarchical Risk Parity, Mean-Variance Optimization и др.).

## MCP-серверы для AI-ассистента

Для работы ИИ как полноценного Quant-разработчика необходимо подключить четыре MCP-сервера:

| Сервер | Назначение |
|---|---|
| `mcp-server-filesystem` | Создание файлов проекта, сохранение данных и графиков |
| `mcp-server-fetch` | Загрузка документации и HTTP-запросы к финансовым API |
| `mcp-server-brave-search` | Поиск научных статей (HRP, SSRN) и тикеров ETF |
| `mcp-server-jupyter` | Исполнение Python-кода, построение корреляций и дендрограмм |

**Быстрый старт:**

```bash
# Скопируйте шаблон конфигурации и заполните своими путями и ключами
cp .mcp.json.example .mcp.json
```

Шаблон конфигурации: [`.mcp.json.example`](.mcp.json.example) (`.mcp.json` с реальными ключами добавлен в `.gitignore`)

Подробные инструкции по установке и настройке каждого сервера: [docs/mcp-setup.md](docs/mcp-setup.md)
